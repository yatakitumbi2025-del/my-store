Security

Each user's data lives at stores/{uid} in Firestore. Make sure your Firestore rules restrict reads and writes to the signed-in owner only: 
rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /stores/{uid} {
      allow read, write: if request.auth != null
                          && request.auth.uid == uid;
    }
  }
}
